# Industrial Maintenance Manager

Java application for basic industrial machine management.

This project was developed to practice Object-Oriented Programming by applying Java concepts to a context related to industrial maintenance and engineering.

---

## Features

✅ Register industrial machines

✅ List registered machines

✅ Search machine by ID

✅ Update machine information

✅ Remove machines

✅ Interactive console menu
